概要

使用本服务须对参数值进行urlencode编码

POI（地点）搜索

服务地址

http://apis.map.qq.com/uri/v1/search    //Web/Wap调用地址
qqmap://map/search    // Android 和 IOS 调用地址

参数说明

参数 示例 说明 必填 web 触屏 android IOS
keyword keyword=麦当劳 搜索关键字 Y        
region（新版支持city、region两种参数名） region=北京 限定搜索城市 N        
bound bound=39.907293,116.368935,39.914996,116.379321 限定搜索坐标范围：
矩形范围区域，参数需给定矩形 左下(西南) 和 右上(东北) 位置坐标
lat,lng<左下角>,lat,lng<右上角> 		N        
coord_type coord_type=1 坐标类型，取值如下：
1 gps
2 gcj02（默认）
如果用户指定该参数，则uri服务需要先进行坐标的转换，再转给Web和wap端，手机地图客户端没有经过服务中转，需要直接支持		 N        
referer referer=可口可乐 调用来源，为了保障对您的服务，请务必填写！ Y        

调用示例

http://apis.map.qq.com/uri/v1/search?keyword=鼓楼&region=北京
//链接打开腾讯地图（自动识别Web/Webapp平台），并显示在 [北京] 范围内的 [鼓楼] 的搜索结果，keyword支持输入公交、地址

qqmap://map/search?keyword=小肥羊&bound=39.907293,116.368935,39.914996,116.379321
//移动端启动腾讯地图App，并在地图中显示 [指定坐标(北京西单)] 范围内的 [小肥羊] 的搜索结果

功能参数值：CurrentLocation ：使用定位点作为起点坐标（仅用于移动端）		 必填
其一

tocoord tocoord=40.010024,116.392239
tocoord=CurrentLocation		 终点坐标：终点坐标优先级高于终点名称
功能参数值：CurrentLocation ：使用定位点作为终点坐标（仅用于移动端）		 必填

Marker（标注）

服务地址

http://apis.map.qq.com/uri/v1/marker    //Web/Wap调用地址
qqmap://map/marker    // Android 和 IOS 调用地址

参数说明

参数 示例 说明 必填 web 触屏 android IOS
coord_type coord_type=1 坐标类型，取值如下：
1 gps
2 gcj02（默认）
如果用户指定该参数，则uri服务需要先进行坐标的转换，再转给Web和wap端，手机地图客户端没有经过服务中转，需要直接支持		 N        
marker   marker参数用于定义具体标注的属性，如标注标题，这类参数有coord、title、addr、tel、uid、pano，marker参数格式：location1|location2|...，
每个location就是一个marker，有一些子参数，具体格式为key:value;key:value;key:value.....
（key与value间用英文冒号分隔，各子参数间用英文分号分隔）

lat,lng,height<街景可选参数> 		Y        
title:超好吃冰激凌 标注点名称 Y        
addr:北京市手帕口桥北铁路道口 地址 N        
tel:010-88888888 电话 N        
uid:11031837053015339230 该参数将指定一个POI的唯一标识，可根据id获取腾讯基础库中的POI详情 N        
pano:10011005120205135839100 街景ID，用于显示标注信息窗口中的街景链接 N        
referer referer=可口可乐 调用来源，为了保障对您的服务，请务必填写！ Y        
bound bound=40.003583,81.323414,
46.143583,87.343414		 底图展示范围 N        

调用示例

http://apis.map.qq.com/uri/v1/marker?coord_type=1&marker=coord:39.892326,116.342763;title:超好吃冰激凌;addr:手帕口桥北铁路道口|coord:40.892326,115.342763;title:超好吃冰激凌;addr:手帕口桥北铁路道口
//链接打开腾讯地图（自动识别Web/Webapp平台），并在指定的几个坐标位置显示Marker，同时打开第一个marker的infowindow

qqmap://map/marker?coord_type=1&marker=coord:39.892326,116.342763;title:超好吃冰激凌;addr:手帕口桥北铁路道口|coord:40.892326,115.342763;title:超好吃冰激凌;addr:手帕口桥北铁路道口
//移动端启动腾讯地图App，并在坐标位置显示Marker

参数说明

参数 示例 说明 必填 web 触屏 android IOS

name name=济乐高速 指定地点的名称 Y

poiId poiId=9193730358662020912 指定地点的poiId N

centerLng centerLng=117.157180 地图中心点的经度 Y

调用示例

lubao://locationByPOI?poiId=9193730358662020912&latitude=37.840420&longitude=117.095460¢erLat=37.309040¢erLng=117.157180&scale=14&name=济乐高速&address=山东省德州市乐陵市
